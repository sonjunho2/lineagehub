# 테스트 방법

## 1. 기본 화면 테스트
- `/` 홈 접속
- `/items` 아이템 목록 확인
- `/monsters` 몬스터 목록 확인
- `/prices` 서버 변경 시 차트 갱신 확인
- `/community` 보드 전환 확인

## 2. 회원 기능 테스트
- 회원가입
- 로그인/로그아웃
- 마이페이지 접근

## 3. 러쉬 테스트
- 일반회원 로그인
- `/simulator` 실행
- 포인트 차감 확인
- 실행 이력 생성 확인

## 4. 관리자 테스트
- `admin / admin1234` 로그인
- `/admin` 접속
- 광고 슬롯 수정
- 회원 포인트 수정
- 아이템 신규 등록

## 5. 운영 점검 항목
- 광고 코드 삽입 후 화면 깨짐 여부
- 세션 유지 여부
- 관리자 권한 없는 사용자의 `/admin` 접근 차단 여부
- 커뮤니티 스팸 방지 정책 추가 여부
