# 2차 버전 확장 설계

## 추가 반영 범위
- 마법 DB
- 스킨/혈마크 자료실
- 신고/제재 시스템
- 포인트 기반 러쉬 실행
- 광고 슬롯 관리자
- CSV 대량 업로드

## 단일 서버 구조
- FastAPI: 사이트/관리자/API/세션 처리
- SQLite: 초기 운영용
- 정적 리소스: app/static
- 템플릿: app/templates

## 운영 시 다음 단계 권장
- SQLite -> PostgreSQL 전환
- 이미지 업로드 저장소 도입
- 관리자 권한 세분화
- 게시글/댓글 삭제 로그, 제재 해제 로그 추가
- 스팸 필터, 욕설 필터, IP 제한 추가
