# 서버 구조

단일 FastAPI 서버에서 다음을 모두 처리한다.
- 공개 웹사이트 렌더링
- 회원 인증
- 관리자 페이지
- 검색 API
- SQLite 데이터 저장

## 디렉토리
- `app/main.py` : 라우트, 인증, 뷰 연결
- `app/database.py` : SQLite 스키마와 시드
- `app/templates` : 공개/관리자 HTML 템플릿
- `app/static` : CSS
- `data/app.db` : 운영 데이터
- `docs/*` : 운영 문서

## 운영 확장 포인트
- Cron으로 시세 배치 수집
- Redis 캐시 추가
- 이미지 업로드 스토리지 연동
- 관리자 로그, 신고관리, 금칙어 필터 추가
