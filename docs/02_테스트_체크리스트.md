# 테스트 체크리스트

1. 홈 접속
2. 아이템 페이지에서 이미지/이름/등급 표시 확인
3. 아이템 상세에서 서버별 최신가 표시 확인
4. 시세 페이지에서 서버 변경 시 목록 변경 확인
5. 회원가입/로그인 확인
6. 커뮤니티 글쓰기/댓글 확인
7. 러쉬 시뮬레이터 실행 후 포인트 차감 확인
8. 관리자 로그인 후 아래 확인
   - 아이템/몬스터/마법/스킨/혈마크 등록
   - 시세 추가 후 공개 시세 페이지 반영
   - 회원 포인트 변경
   - 회원 상태 변경
   - 게시글 숨김 처리
   - 신고 상태 변경
   - 광고 슬롯 수정
