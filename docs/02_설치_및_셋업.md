# 설치 및 셋업 방법

## 1. 준비
- Python 3.11 이상 권장
- 기본 포트: 3000

## 2. 실행
```bash
python run.py
```
초기 실행 시 `data/app.db`가 자동 생성되고 샘플 데이터가 시드된다.

## 3. 운영 전 변경해야 할 것
- `app/main.py` 세션 시크릿 변경
- `settings` 테이블의 `adsense_client`, `analytics_id` 실제 값 입력
- 관리자 계정 비밀번호 변경
- 샘플 데이터 제거 후 실제 데이터 입력

## 4. 광고 코드 적용
관리자 > 광고 슬롯 관리에서 각 슬롯에 실제 AdSense 코드를 입력한다.
권장 슬롯:
- home_top
- home_mid
- db_sidebar
- community_inline
- simulator_result

## 5. 포인트 정책
기본값: 러쉬 1회 100포인트 차감
변경 위치:
- DB `settings` 테이블의 `point_cost_simulation`

## 6. 실데이터 투입 권장 순서
1. 아이템 DB 정리
2. 몬스터 드랍 정리
3. 서버 시세 수집 배치 작성
4. 커뮤니티 운영정책 수립
5. 광고 승인 후 광고 코드 적용
