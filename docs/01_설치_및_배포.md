# 설치 및 배포

## 로컬 실행
1. 압축 해제
2. 터미널에서 프로젝트 폴더 열기
3. 아래 실행
```bash
pip install -r requirements.txt
python run.py
```
4. 브라우저에서 `http://localhost:3000`

## Render
- Build Command: `pip install -r requirements.txt`
- Start Command: `python run.py`
