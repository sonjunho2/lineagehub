# Lineage Classic Hub

리니지클래식 통합형 팬사이트 운영 베이스 프로젝트.

## 포함 기능
- 아이템 DB / 몬스터 DB / 마법 DB
- 시세 페이지
- 포인트 기반 러쉬 시뮬레이터
- 스킨 / 혈마크 자료실
- 커뮤니티
- 신고 / 제재 / 회원 상태 관리
- 광고 슬롯 관리
- CSV 대량 업로드

## 실행
```bash
pip install -r requirements.txt
python run.py
```

브라우저: http://localhost:3000

기본 계정
- admin / admin1234
- tester / user1234

## 주의
이 프로젝트는 참고 사이트의 UX 범주를 참고해 구조를 설계한 운영 베이스다.
타 사이트의 원문, 이미지, DB를 무단 복제하는 기능은 포함하지 않는다.
실데이터는 data/import_templates 예시에 맞춰 운영자가 적재해야 한다.
