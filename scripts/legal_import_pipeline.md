# 실데이터 적재 파이프라인

이 프로젝트에는 타 사이트를 무단 복제하는 크롤러를 포함하지 않는다.
대신 다음 두 방식으로 데이터 적재를 지원한다.

1. 운영자가 직접 확보한 합법적 원본을 CSV 템플릿에 맞춰 정리 후 관리자 업로드
2. 공식 API, 공개 데이터, 직접 제작 이미지처럼 사용 권한이 명확한 자료만 별도 ETL 스크립트로 변환

## 권장 절차
- data/import_templates/*.csv 형식에 맞춰 자료 정리
- /admin 로그인
- CSV 일괄 업로드 사용
- DB 반영 후 상세 페이지/검색/시세/시뮬레이터에서 검수
